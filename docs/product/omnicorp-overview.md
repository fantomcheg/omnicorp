# OmniCorp

Мой план действий:
1. Сначала - структура и Git (30 минут)
Создать базовую структуру папок (frontend/, backend/, labs/, docs/)
Инициализировать Git репозиторий
Создать .gitignore, README.md
Загрузить на GitHub
2. Затем - минимальный Frontend MVP (1-2 часа)
Создать React проект с PixiJS
Реализовать простую сцену (фон + спрайт персонажа)
Добавить движение WASD
Это даст нам визуальный результат и мотивацию
3. Параллельно - базовый Backend (30 минут)
Простой Node.js + Express сервер
Эндпоинт /health
Готовность к интеграции с фронтендом
Почему именно такой порядок:
✅ Быстрый визуальный результат - увидим игру в действии
✅ Проверим совместимость React + PixiJS
✅ Заложим основу для дальнейшего развития
✅ Следуем roadmap - это именно то, что запланировано на первые недели
Начнем с создания структуры проекта? Это займет буквально 10-15 минут, и у нас будет четкая основа для дальнейшей работы.
