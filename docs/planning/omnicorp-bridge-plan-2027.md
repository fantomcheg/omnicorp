# Bridge Plan 2027 — OmniCorp Game Project

## Фаза 1. Подготовка (январь–март 2027)

- [ ] Освежить **Electron** (десктоп-фреймворк).
- [ ] Освежить **PixiJS/Phaser.js** (2D-графика).
- [ ] Переписать старые наработки OmniCorp под новый стек.
- [ ] Настроить GitHub репозиторий и CI/CD (GitHub Actions + Docker).
- [ ] Создать MVP-скелет игры (окно, сцена, меню, первый модуль).

## Фаза 2. Минимальный игровой прототип (апрель–июнь 2027)

- [ ] Реализовать систему сцены (карта, движение, UI).
- [ ] Встроить базовый геймплей (миссия «взлом веб-приложения»).
- [ ] Добавить симуляцию XSS и SQLi (мини-лабы).
- [ ] Подключить минимальные ассеты (UI + спрайты).
- [ ] Сделать playable demo.

## Фаза 3. Расширение (июль–сентябрь 2027)

- [ ] Добавить multiplayer/online (Express backend + WebSockets).
- [ ] Реализовать AI для NPC (ML-навыки, адаптивные противники).
- [ ] Доработать UI/UX (React + Tailwind для меню).
- [ ] Подготовить Early Access билд (Steam или itch.io).
- [ ] Собрать версию «0.5» с сюжетом и несколькими миссиями.

## Фаза 4. Публикация (октябрь–декабрь 2027)

- [ ] Запустить сайт и devlog (расширение личного блога под игру).
- [ ] Сделать локализацию (EN + RU).
- [ ] Проверить безопасность кода (применить свои навыки пентеста).
- [ ] Провести маркетинговую кампанию (Twitter, Reddit, Dev.to).
- [ ] Выпустить OmniCorp v1.0 в Early Access.
